# homebrew-tap

Personal Homebrew tap.

| Formula | Project |
|---|---|
| `note` | [LLawli/note](https://github.com/LLawli/note) — take notes entirely from the terminal (CLI + TUI) |
| `xclaude` | [LLawli/XClaudeDashboard](https://github.com/LLawli/XClaudeDashboard) — real-time TUI dashboard for [XClaudeUsage](https://github.com/SrDarf/XClaudeUsage) |

## Usage

```sh
brew install LLawli/tap/note
brew install LLawli/tap/xclaude
```
